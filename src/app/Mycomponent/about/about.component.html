Welcome to TodoApp

TODOSAPP provides an easy feature to mark your daily activities which makes you remember every time, indeed which reminds your schedule.

About the app: By clicking on the Add todo button you todo activity will be displaying on to the page.By marking the Done Chekbox you can understand that the work is done. Also you can delete your todo by clicking on the Delete button.

This Application is build using Angular as Front End Framework and back-end database as the InternalStorage of a PC/device. It uses a complete stack of Angularjs and it is hosted freely using github pages .
